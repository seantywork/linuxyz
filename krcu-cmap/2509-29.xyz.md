# 03

```shell

|---(r)eader thread------------|---(w)riter thread-----------|
|    wait for cmap to be ready |                             |
|                              |   create cmap               |
|                              |   update cmap (fill up)     |
|                              |   ready!                    |
|                              |                             |
|    read values from cmap     |   update cmap               |
|    (interval 100ms)          |   (interval 1000ms)         |
|    continue                  |   continue                  |
|    continue                  |   continue                  |
|    continue                  |   continue                  |
|    continue                  |   delete all nodes          |
|    continue...               |   exit                      |
|    untile there is no value  |                             |
|    (sum == 0)                |                             |
|    exit                      |                             |
--------------------------------------------------------------
```
